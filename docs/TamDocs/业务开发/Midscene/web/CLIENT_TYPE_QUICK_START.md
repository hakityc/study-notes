# 客户端类型快速开始

## 🚀 30 秒快速上手

### 1. 打开调试页面

访问：`http://localhost:5173`

### 2. 找到"客户端类型"选择器

在页面顶部的"消息元数据"区域，你会看到：

```
┌──────────────────────────┐
│ 客户端类型 📱           │
│ ┌──────────────────────┐ │
│ │ 自动（Web） ▼       │ │  ← 点击这里
│ └──────────────────────┘ │
└──────────────────────────┘
```

### 3. 选择目标平台

- **📱 自动（Web）** - 测试浏览器网页（默认）
- **📱 Web 端** - 明确指定 Web 平台
- **🖥️ Windows 端** - 测试 Windows 桌面应用

### 4. 发送消息

选择平台后，输入指令并点击"发送消息"！

## 📋 使用示例

### Web 端（浏览器）

```
1. 选择：📱 Web 端
2. 输入：点击搜索按钮
3. 点击：发送消息
✅ 在浏览器中执行操作
```

### Windows 端（桌面）

```
1. 选择：🖥️ Windows 端
2. 输入：打开记事本
3. 点击：发送消息
✅ 在 Windows 桌面执行操作
```

## ⚡ 关键点

1. **默认是 Web 端** - 不选择时自动使用 Web
2. **图标提示** - 📱 = Web，🖥️ = Windows
3. **即时切换** - 随时更改目标平台
4. **JSON 预览** - 右侧面板显示完整消息

## 🎯 常见操作

| 想做什么 | 选择哪个 |
|---------|---------|
| 操作网页 | 📱 Web 端 |
| 打开 Windows 应用 | 🖥️ Windows 端 |
| 填写表单 | 📱 Web 端 |
| 文件操作 | 🖥️ Windows 端 |
| 网页脚本 | 📱 Web 端 |
| 桌面自动化 | 🖥️ Windows 端 |

## 💡 提示

- **看不到 Windows 选项？** 可能是浏览器缓存，刷新页面试试
- **Windows 端无响应？** 确保 Windows 客户端已连接
- **不确定选哪个？** Web = 浏览器，Windows = 桌面应用

就这么简单！🎉
